    🇸🇪 Kurserna (🇬🇧 The courses)¶

    Dessa är kurserna vi lär ut under Lördagskurserna:

    Kurs Tid Vad Bok
    Arduino 10:00-12:00 Ett kretskort som programmeras för att bygga maskiner Arduino för ungdomar
    Blender 10:00-12:00 Ett 3D ritnings program för, t.ex. 3D skrivning Grundkurs i Blender
    Programmering 12:00-14:00 Skapa datorspel, med bland annat, programmerspråket Processing Processing för ungdomar

    Varje tillfälle du deltar vid får du välja en av kurserna för det tillfället.

    Se ditt första besök hur ditt första besök kommer ske!

    Om du är ännu mer avancerade, kann du också göra:

    Kurs Bok
    git git for youngsters
    laserskärare laser cutter
    vinylskärare vinyl cutter

    These are the courses we teach at the Saturday courses:

    Course Time What Book
    Arduino 10:00-12:00 A programmable printed circuit board to build electronical machines with Arduino för ungdomar
    Blender 10:00-12:00 A program to create 3D models for, among others, 3D printing Grundkurs i Blender
    Programmering 12:00-14:00 Create games with, among others, the programming language Processing Processing för ungdomar och git for youngsters

    Each time you come in, you get to pick a course to follow that day.
